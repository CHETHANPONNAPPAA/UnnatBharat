## 📦 Prerequisites
- Python 3.8 or higher
- Git
- pip package manager
- Dependencies listed in `requirements.txt`

## 🚀 Installation

```bash
# Clone the repository
git clone <repository-url>
cd UnnatBharat

# Install required dependencies
pip install -r requirements.txt
```
